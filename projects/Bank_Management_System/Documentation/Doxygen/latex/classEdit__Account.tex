Checks the authorization of the customer based on their credentials. 

This function verifies whether the provided name, national number, and password match the stored information in the database. If the credentials are valid, it returns the customer ID. If any field is empty or incorrect, appropriate messages are displayed, and the function returns -\/1.

\begin{DoxyReturn}{Returns}
int The customer ID if authorization is successful, otherwise -\/1.
\end{DoxyReturn}
The function performs the following checks\+:
\begin{DoxyItemize}
\item Validates that the name, password, and national number fields are not empty.
\item Retrieves the customer\textquotesingle{}s information from the database.
\item Compares the entered credentials with the retrieved information.
\end{DoxyItemize}

If the entered information matches, the customer ID is returned. If not, specific warning messages are displayed for incorrect fields.
