\doxysubsection{Detailed Description}
Dialog for customer operations such as deposit, withdraw, and balance inquiry. 

This class handles the UI and operations related to customer transactions, including depositing funds, withdrawing funds, and checking account balance.

Performs a banking operation based on the specified operation type. 

This function retrieves user input for the customer\textquotesingle{}s name, national number, password, and the amount for the transaction. It validates the inputs and checks if an account exists with the provided national number. Depending on the operation type (deposit, withdraw, or balance enquiry), it executes the corresponding action.
